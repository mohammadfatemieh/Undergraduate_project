## 项目五、机器人的舵机控制板设计项目

* 该项目主要是编写BOOTLOADER代码实现

* 控制板支持体操等竞步机器人的在线动作调试

* 支持无线的机器人动作调试

## 项目六、矿下受力监测项目

主要实现功能：

* 实现钢架应力的精确采集
* 精确测出钢架梁的受力大小，如果受力过大 就进行报警
* 数据通过以太网上传至服务其中 
* 数据存储方拾贰，存到硬件设备的内存卡中
* 报警功能
